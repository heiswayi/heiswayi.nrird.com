Jekyll Themes

Jekyll is a popular static site generator written in Ruby. Over the past years, I've created some of minimalist Jekyll themes and published their source code on GitHub. This site also is built and compiled using Jekyll engine to become a static site.

The Plain
Center aligned. Single column layout. Ideal for personal blogging purpose.
GitHub · Demo
Thinkspace
Center aligned. Single column layout. Ideal for technical writing style blog.
GitHub · Demo
TextLog
Left aligned. Single column layout. No site menu in post layout. Ideal for logging notes or technical documentations.
GitHub · Demo
Barebone
The most barebone Jekyll theme. No Sass/SCSS file. Run with almost pure HTML.
GitHub · Demo
